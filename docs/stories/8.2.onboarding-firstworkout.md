# Story 8.2: Onboarding - Premier workout guidé

## Status
**Draft**

---

## Story

**As a** nouvel utilisateur,  
**I want to** être guidé pour créer mon premier workout,  
**so that** je démarre facilement.

---

## Acceptance Criteria

1. Après onboarding, prompt "Create your first workout"
2. Option rapide: "Use a template" (Show templates)
3. Option custom: "Create from scratch"
4. Guide user à ajouter au moins 3 exercices
5. Success message après premier workout créé
