# Story 7.2: Settings - Dark Mode

## Status
**Draft**

---

## Story

**As a** utilisateur,  
**I want to** choisir entre Dark Mode et Light Mode,  
**so that** l'app est confortable selon mon environnement.

---

## Acceptance Criteria

1. Option "Theme": System, Light, Dark
2. Préférence sauvegardée
3. Appliquée instantanément
4. Default: System (suit le système)
