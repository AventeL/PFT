# Story 8.1: Onboarding - Tutorial interactif

## Status
**Draft**

---

## Story

**As a** nouvel utilisateur,  
**I want to** un tutorial rapide au premier lancement,  
**so that** je comprends comment utiliser l'app.

---

## Acceptance Criteria

1. Onboarding au premier lancement uniquement
2. 3-4 screens max (< 2min)
3. Couvre: Create workout, Start workout, Track sets, Timer
4. Skip option disponible
5. Marqueur "onboarding_completed" (SharedPreferences)
