## Development Workflow

1. **PM creates story** from PRD using `*create-story` command
2. **Story moves to "Draft"** status
3. **Dev reviews story** and moves to "In Progress"
4. **Dev implements** following tasks/subtasks
5. **Dev updates** "Dev Agent Record" section
6. **QA validates** and fills "QA Results"
7. **Story moves to "Done"**
